How to install:

  • 1. Unpack the MASh zip archive on your local hard drive ( via winrar )
  • 2. Upload the theme to the themes folder on your server ( via filezilla )
  • 2. Upload the recommended plugins to the plugin folder
  • 4. Go to Admin WordPress and select Design
  • 5. Select the MASh WordPress Theme
  • 6. Configure via the WP Settings tab : flickrRSS, Twitter and Feed Count
  • 7. You're done, have fun.

Recommended Plugins:

  • 1. WP Cache available at Uib
  • 2. Poll/Page Navi Plugin available at Lester Chan
  • 3. Twitter messages available at Jinlabs
  • 4. SecurityScan available at SecurityScan
  • 5. Login Lockdown available at Login Lockdown

Customization:

  • - Open the theme folder /res/ and and create your own background images
  • - Alignment classes: by default no post images are floated
    • use alignleft, alignright or align center
  • - configure the Flickr admin in the MASh options panel: find your Flickr ID
  • (should look like this:1234567@N00) and paste it inside the panel
  • - Open blurb.php, edit your Twitter username
  • Note: Your tweets must be unprotected (i.e. in the public timeline)
  • - Open /inc/ads.php, edit the Advertising address and images
  • - Open /inc/spo.php, edit the Advertising address and images
  • - by default various areas are widgetized, like sidebar, top blue panel, just drop your widgets
  • - various custom widgets are onboard, like search, about, most comments and categories
  • - The MAShtheme is style switcher enabled, if you use widgets in the blue header, then the switcher will disappear
  • - Support for WP 2.5.* tags is built in, just tag your posts

Thumb image

  • 1. Find an appropriate image, resize it (max 320px width) and upload it through the post admin panel.
  • 2. Copy the image URL
  • 3. Scroll in post admin panel to CUSTOM FIELDS, select KEY from left menu
  • 4. Add the KEY "thumbnail" into the KEY field
  • 5. Paste the image URL into the VALUE field
  • http://PATH TO IMAGE/uploads/2007/09/vj.gif
  • 6. Save
  • 7. Check your post for errors, if no errors "Publish".
